# Abdulmajeed Alqithmi Portfolio

Production-ready portfolio website for Abdulmajeed Alqithmi, built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Lucide React.

## Commands

```bash
npm install
npm run build
```

## Editing Content

Most portfolio content lives in `lib/data.ts`. Replace placeholders there as your experience, projects, links, and resume details evolve.
